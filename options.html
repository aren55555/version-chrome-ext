Version Extension

How it works

This extension detects git commit SHAs from web pages and creates direct links to view them on GitHub.

Supported sources

  • JSON pages: Uses JSONPath to find the git SHA in JSON content
  • HTML pages: Looks for a meta tag with the configured name

How to use

  1. Add a GitHub repository URL
  2. Add one or more URL matchers for that repo
  3. Visit a page that matches your pattern
  4. Click the extension icon to see the commit link

The GitHub URL will be constructed as:
[repo]/commit/[version]
