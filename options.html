URL to GitHub Repository Mappings

Configure which GitHub repository to use for specific URLs or URL patterns.

Match URLs containing this text (e.g., "api.pallet.com" matches "https://api.pallet.com/version")
Example: https://github.com/octocat/Hello-World

How it works:

This extension detects git commit SHAs from:

  • JSON pages: Looks for a "version" field in the JSON
  • HTML pages: Looks for a meta tag with the configured name

When a version is detected:

  1. The current URL is matched against your configured patterns
  2. A link is created to view that commit on the matching GitHub repository
  3. The link is displayed in the extension popup

The GitHub URL will be constructed as: [matching-repo]/commit/[version-value]
