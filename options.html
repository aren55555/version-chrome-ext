Version Extension Settings

URL to GitHub Repository Mappings

Configure which GitHub repository to use for specific URLs or URL patterns.

Match URLs containing this text (e.g., "api.pallet.com" matches "https://api.pallet.com/version")
Example: https://github.com/octocat/Hello-World

How it works:

When you visit a page that contains JSON with a "version" field, this extension will:

  1. Detect the version value (assumed to be a git commit SHA)
  2. Match the current URL against your configured patterns
  3. Create a link to view that specific commit on the matching GitHub repository
  4. Display the link in the extension popup

The GitHub URL will be constructed as: [matching-repo]/commit/[version-value]

Example: Pattern "api.pallet.com" will match any URL containing that text, like "https://api.pallet.com/version"
